CONTENTFUL FILE UPLOADER

Python executable for uploading files to contentful.


HOW TO USE:

	1. Make sure within upload folder are all your files you with to upload to contentful
	2. Double click on contenful executable
	3. Google chrome will open and prompt you to login.
	4. Login and navigate to page where you wish to upload your content
	5. Return to new terminal screen and hit enter, program will upload files to their corresponding tag
	6. Publish content, you can then either close application (which will close new chrome page) or go to a new page and upload content again, repeating steps 4-6
